# TuxSay

**TuxSay** is a simple command-line utility that displays messages in an ASCII art speech bubble, optionally using different ASCII art characters. It also has an option to display random Linus Torvalds quotes.

## Features

- Display custom messages with an ASCII art character.
- Show random Linus Torvalds quotes.
- Choose from multiple ASCII art characters.

## Usage

```bash
tuxsay [--help | --fortune | <message>] [<character>]
```

## Options

--help:
    Displays the help message with usage instructions.

--fortune:
    Displays a random quote from the quotes.txt file in a speech bubble with the specified ASCII art character.

\<message>:
    The message to be displayed inside the speech bubble.

\<character>:
    The name of the ASCII art file to use (without the .txt extension). If not specified, the default character (tux) is used. The characters that come with this program are tux, shrek, pikachu, crying, chad.

## Compiling
```
gcc -o tuxsay tuxsay.c
```

## Customization
You can add new ASCII characters

To add new ASCII characters:

- Create a .txt file with your ASCII art inside the characters/ directory.
- Name the file appropriately (e.g., alien.txt).
- The ASCII art can be used by specifying the filename (without .txt) as the <character> argument.
